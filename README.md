# dotfiles

## Windows

`git clone git://github.com/pogopaule/dotfiles.git`<br/>
`cd dotfiles`<br/>
`install.bat`<br/>

## Mac and Linux

`git clone git://github.com/pogopaule/dotfiles.git; cd dotfiles; ./install.sh`

## Install new vim plugin

1. Add to vimrc
2. Run `:PlugInstall` in vim

## Update vim plugins

Run `:PlugUpdate` in vim
